## set os attr

### Usage

`stack set os attr {os ...} {attr=string} {value=string} [shadow=boolean]`

### Description


	Sets an attribute to an os and sets the associated values

	

### Arguments

* `[os]`

   Name of os


### Parameters
* `[attr=string]`
* `[value=string]`
* `{shadow=boolean}`

   If set to true, then set the 'shadow' value (only readable by root
	and apache).
